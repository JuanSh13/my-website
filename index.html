¿Quién Soy?

Soy estudiante de 8° semestre en Ingeniería en Sistemas con más de 3 años de experiencia como Desarrollador Full Stack. Me especializo en desarrollo First Mobile y optimización SEO, con solidas habilidades en JavaScript, Python y PHP, así como en frameworks como Angular y React. Aplico prácticas SOLID y Clean Code para crear soluciones eficientes mediante el consumo de API REST y la integración de microservicios. Trabajo con metodologías ágiles como Scrum, lo que me permite entregar proyectos escalables y alineados con las necesidades del cliente. Además, tengo experiencia en diseño UX/UI, asegurando accesibilidad y usabilidad en mis desarrollos.

¿Qué Puedo Hacer?

Diseños Creativos y Adaptativos

Crear soluciones visuales atractivas y originales que se ajusten de manera efectiva a diferentes dispositivos y entornos. Combinando la creatividad con un sólido conocimiento de los principios de diseño para transmitir mensajes de manera efectiva.

UX - UI

Diseñar y desarrollar interfaces que sean fáciles de usar y visualmente atractivas, lo que proporciona y abarca una experiencia satisfactoria a la mayoria de usuarios posibles.

Uso de microservicios y API REST

Implementar microservicios y API REST para crear aplicaciones escalables y modulares, mejorando la eficiencia y mantenibilidad del sistema. Esta arquitectura permite integrar funcionalidades de manera independiente, optimizando el rendimiento y la seguridad, mientras se satisfacen las necesidades del cliente.

Control de Versiones con Git

Uso de Git para gestionar y controlar los cambios en un proyecto de desarrollo de software. Esto implica el seguimiento de las modificaciones realizadas en los archivos del proyecto, la gestión de ramas (branches) para trabajar en paralelo, la fusión de cambios, la gestión de conflictos y la colaboración con otros desarrolladores.

Código Limpio y buenas prácticas

Buenas prácticas de código, que se adecua a la facilidad de entender, mantener y modificar, proporcionando una mejor legibilidad y compresión coherente, así optimizar la eficiencia del avance del proyecto y la comunicación colaborativa.

Desarrollador Back-end

Mantenimiento y construcción de la lógica y la funcionalidad que se promueve detrás del sistema, garantizando que la aplicación funcione correctamente y se comunique eficientemente con el Front-End.
